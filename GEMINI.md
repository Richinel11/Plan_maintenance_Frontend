# ENEOPLAN — Projet de design sur Google Stitch

## Étape 1 — Configurer le projet
Utilise le projet Stitch 17940002581651913904.

## Étape 2 — Créer le système de design
Crée un système de design pour le projet ENEOPLAN avec les paramètres suivants :
- Couleur primaire : #1B75BB
- Couleur secondaire : #14689E
- Couleur accent : #8DC640
- Couleur texte principal : #1C1C1B
- Couleur texte secondaire : #939597
- Couleur fond : #FFFFFF
- Police : Poppins ou Inter
- Coins : légèrement arrondis (8px)
- Mode : clair uniquement

Applique immédiatement ce système de design au projet après sa création.

## Étape 3 — Générer les écrans

Génère les écrans suivants dans l'ordre indiqué en utilisant le système de design défini.

- Écran 7 : Formulaire de création d'un travail — Étape 2 (Localisation & Consistance)
- Écran 8 : Formulaire de création d'un travail — Étape 3 (Programmation & Impact)

- Écran 13 : Calendrier (vues Mois / Semaine / Jour / Liste)
- Écran 14 : Page liste des alertes de conflits
- Écran 15 : Page plannings effectifs

### PROFIL : AUDITEUR / PLANIFICATEUR

- Écran 32 : Page export des données

## Étape 4 — Appliquer le système de design
Après génération de tous les écrans, applique le système de design 
ENEOPLAN à l'ensemble des écrans du projet pour garantir une cohérence 
visuelle parfaite sur toutes les pages.

## Règles générales à respecter sur tous les écrans
- Application web desktop uniquement, largeur 1440px
- Langue : français uniquement
- Header commun : barre de navigation en #1B75BB avec le nom 
  "ENEOPLAN" en blanc à gauche et le logo ENEO en dessous à droite 
  du nom, nom de l'utilisateur connecté et cloche de notification 
  à droite
- Sidebar de navigation à gauche sur toutes les pages 
  (sauf page de connexion)
- Fond de page : blanc #FFFFFF
- Cartes avec ombre légère
- Bouton principal : fond #1B75BB, texte blanc
- Bouton secondaire : contour #1B75BB, texte #1B75BB
- Badges de statut : arrondis en forme de pill
- Champs de formulaire : bordure grise #939597, 
  focus en #1B75BB
- Champs auto-calculés : fond vert clair avec badge "Calculé Auto"
- Champs auto-remplis : fond vert clair avec badge "Auto-rempli"
- Champs obligatoires : astérisque en #8DC640
